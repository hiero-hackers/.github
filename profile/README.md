# Hiero Hackers

Welcome to Hiero Hackers 👋

This GitHub organization is a lightweight playground for the community around [Hiero](https://hiero.org).

Here you can experiment, prototype, and build ideas around the Hiero ecosystem without heavy processes or governance overhead.

If you have an idea for a tool, SDK, demo, integration, or hackathon project — this is the place to build it.

![hiero-community.png](hiero-community.png)

## What is Hiero?

Hiero is an **open-source, vendor-neutral distributed ledger technology hosted by the Linux Foundation**.
It contains the core software used to build and run the Hedera public network and related developer tooling.  ￼

The Hiero ecosystem includes nodes, SDKs, developer tools, and services that enable developers to build decentralized applications and infrastructure.

## What is Hiero Hackers?

Hiero Hackers is intentionally simple and lightweight.

This organization exists so the community can:
- experiment with ideas
- build demos and prototypes
- share examples and tutorials
- collaborate during hackathons
- explore new tools for the Hiero ecosystem

We optimize for speed and creativity, not process.

That means:
- minimal governance
- no heavy branch protections
- no complex workflows
- no bureaucracy

Fast forward > process first.

If an idea works — great!
If not — that’s fine too.

### Want a repository?

Anyone in the community can request a repository.

Just open an issue here:

➡️ https://github.com/hiero-hackers/governance/issues

Tell us:
- the repository name
- a short description
- optionally: additional committers

A maintainer will create the repository for you and give you other committers write rights to it.

### What belongs here?

Examples include:
- hackathon projects
- SDK experiments
- sample applications
- developer tools
- tutorials and example code
- ...

If you’re unsure — just ask in the [governance repo](https://github.com/hiero-hackers/governance).

### Community Principles

A few simple rules:
- Be respectful
- Be constructive
- Share what you learn
- Help others build

Every repo, community and committer needs to follow our Code of Conduct that will be availble in near future.

We follow the general open-source spirit of the Hiero ecosystem.
